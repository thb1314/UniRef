# UniRef-UAV Homepage

This repository currently hosts a static benchmark homepage prototype for **UniRef-UAV: A Multimodal Benchmark for Universal Referring in UAV Imagery**.

## Files

- `index.html`: single-page homepage
- `styles.css`: page styles
- `assets/uniref-overview.png`: introduction figure used as the main visual

## Local Preview

```bash
python3 -m http.server
```

Then open `http://127.0.0.1:8000`.
